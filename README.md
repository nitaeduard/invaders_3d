# invaders_3d

A 3D Space-Invaders–style browser game/demo implemented in TypeScript.

## Summary
invaders_3d is a TypeScript project that implements a classic Space Invaders-style game in 3D. The repository contains the source code for rendering, game logic, input handling and basic UI.

## Features
- Player ship movement and shooting
- Enemy waves and simple AI
- Projectile and collision handling
- Camera and 3D rendering
- Basic UI for score and game state

## Install & Run
1. Install dependencies
   npm install

2. Start development server
   npm run dev

3. Build for production
   npm run build

4. Preview production build locally
   npm run preview

## Usage / Controls
- Move left/right: Arrow keys or A / D
- Shoot: Space
- Pause: P

(Adjust controls above if your project uses different input mappings.)
